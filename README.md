# Bulk GSC Index Checker 🚀

A Chrome extension that allows webmasters and SEO professionals to paste a bulk list of URLs, check their live index status using the Google Search Console (GSC) API, and export the results to CSV files.

## ✨ Features

* **Secure Authentication:** Uses Chrome's built-in OAuth identity API to securely log into your Google Account.
* **Property Selection:** Automatically fetches and displays your verified Google Search Console properties.
* **Bulk URL Inspection:** Paste multiple URLs and check their current index status in real-time.
* **Data Export:** Easily download your results as CSV files (separated by "Indexed" and "Not Indexed").
* **Premium UI:** Features a sleek, responsive interface with a dark mode aesthetic and progress tracking.

## 🛠️ Prerequisites

To use this extension, you must set up your own Google Cloud Project to generate an OAuth Client ID:

1. Go to the [Google Cloud Console](https://console.cloud.google.com/).
2. Create a new project.
3. Go to **APIs & Services > Library** and enable the **Google Search Console API**.
4. Go to **APIs & Services > OAuth consent screen**. Select **External** and fill in your details. Add your email under "Test users".
5. Add the scope: `https://www.googleapis.com/auth/webmasters.readonly`.
6. Go to **Credentials > Create Credentials > OAuth client ID**.
7. Choose **Chrome app/Extension**.
8. Copy the generated Client ID and paste it into the `manifest.json` file of this extension (replacing `YOUR_OAUTH_CLIENT_ID_HERE`).

## 📥 Installation

Currently, this extension is meant to be loaded locally as an unpacked extension:

1. Download or clone this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Turn on **Developer mode** (toggle in the top right corner).
4. Click **Load unpacked** and select the folder containing the extension files.
5. *Optional but recommended:* Copy the Extension ID generated by Chrome and add it to your Google Cloud Console OAuth settings for enhanced security.

## 💻 How to Use

1. Click the extension icon in your Chrome toolbar.
2. Click **Sign in with Google** and authorize the extension.
3. Select the target website property from the dropdown menu.
4. Paste your list of URLs (one per line) into the text area.
5. Click **Check Index Status**.
6. Wait for the progress bar to complete. 
7. Use the Export buttons to download your indexed or non-indexed URLs as a CSV.

## ⚠️ API Limits

Please note that the Google Search Console URL Inspection API has strict quotas:
* **2,000 queries per day** per website property.
* **600 queries per minute** per website property.
Checking massive lists of URLs is subject to these Google-imposed limits.
